# Лабораторна робота №6: C++ openCV

## Опис завдання

Метою лабораторної роботи є розробка інтерактивного застосунку для захоплення та обробки відеопотоку з камери з використанням бібліотеки OpenCV.

Програма повинна забезпечувати:
* отримання відео з камери в реальному часі
* застосування різних методів обробки зображення
* взаємодію з користувачем через клавіатуру, мишу та графічні елементи (trackbar)

## Реалізований функціонал

### Робота з відео
* Захоплення відео з камери (VideoCapture)
* Відображення відеопотоку у вікні
* Виведення FPS у реальному часі

### Режими обробки зображення
Реалізовано декілька режимів:
* Оригінальне зображення
* Відтінки сірого (Grayscale)
* Розмиття (Blur)
* Виявлення границь (Canny)
* Інверсія (Invert)
* Порогова обробка (Threshold)
* Собель-фільтр (Sobel)

### Взаємодія з користувачем

#### Клавіатура
* перемикання режимів обробки (цифри 1-7)
* вихід з програми (клавіша Esc)

#### Миша
* можливість вибору області на зображенні
* відображення прямокутника на кадрі


### Trackbar (слайдери)
* керування параметрами яскравості зображення

### Архітектура програми

Проєкт має модульну структуру:

* `CameraProvider` — робота з камерою
* `FrameProcessor` — обробка кадрів
* `KeyProcessor` — обробка клавіш
* `Display` — відображення результату

## Вимоги до системи

* Операційна система: **Linux / Windows / macOS**
* C++ компілятор з підтримкою C++17
* CMake 3.10+
* OpenCV 4.x
