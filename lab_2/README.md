# Лабораторна робота №2: Наука про дані: підготовчий етап

## Опис завдання

Лабораторна робота складається з двох частин, які знайомлять з базовими прийомами роботи з даними у Python.

* **Частина 1:** Автоматичне завантаження та обробка даних VHI-індексу для областей України з використанням urllib та pandas.

* **Частина 2:** Практичне застосування методів фільтрації, масштабування та кодування даних на прикладі датасету споживання електроенергії.

## Вимоги до системи

* Операційна система: Windows / macOS / Linux
* Версія інтерпретатора: Python 3.9+
* Залежності: (`pandas`, `scikit-learn`, `jupyter` тощо), які вказані у файлі requirements.txt.

## Інструкція із запуску

1. Відкрити термінал у теці lab-2.
2. Створити та активувати віртуальне середовище: `python -m venv venv` `venv\Scripts\activate`
3. Встановити необхідні для виконання роботи залежності командою: `pip install -r requirements.txt`
4. Для виконання другої частини роботи потрібно завантажити файл household_power_consumption.txt із сайту https://archive.ics.uci.edu/dataset/235/individual%2Bhousehold%2Belectric%2Bpower%2Bconsumption та розташувати у теці lab-2.
5. Запустити Jupyter Notebook в терміналі: `jupyter notebook`
6. Відкрити файли part1.ipynb та part2.ipynb та виконайти комірки із кодом.
