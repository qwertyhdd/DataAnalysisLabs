# Лабораторна робота №5: Наука про дані: обмін результатами та початковий аналіз

## Опис завдання

Метою лабораторної роботи є створення інтерактивного веб-додатку для аналізу та візуалізації даних за допомогою бібліотеки Streamlit.

У роботі використовується датасет із лабораторної роботи №2, що містить часові ряди:

* VCI (Vegetation Condition Index)
* TCI (Temperature Condition Index)
* VHI (Vegetation Health Index)

Додаток дозволяє користувачу взаємодіяти з даними, застосовувати фільтри та аналізувати результати у зручному вигляді.

## Реалізований функціонал

* Dropdown список для вибору індексу
    * VCI / TCI / VHI
* Dropdown список для вибору області
* Slider для вибору діапазону тижнів
* Slider для вибору діапазону років
* Checkbox
    * сортування за зростанням
    * сортування за спаданням
    * якщо обрані обидва → сортування не застосовується
* Кнопка Reset
    * повертає всі фільтри до початкового стану

## Вимоги до системи

* Операційна система: Windows / macOS / Linux
* Версія інтерпретатора: Python 3.9+
* Залежності: (`streamlit`, `pandas`, `numpy` тощо), які вказані у файлі requirements.txt.

## Інструкція із запуску

1. Відкрити термінал у теці lab-5.
2. Створити та активувати віртуальне середовище: `python -m venv venv` `venv\Scripts\activate`
3. Встановити необхідні для виконання роботи залежності командою: `pip install -r requirements.txt`
4. Запустити програму `streamlit run lab5.py`. (В разі не відкриття сторінки в браузері автоматично потрібно перейти за посиланням http://localhost:8501)
