# Лабораторна робота №3: Візуалізація даних

## Опис завдання

Метою лабораторної роботи є ознайомлення з основними методами візуалізації даних та аналізу багатовимірних датасетів за допомогою Python.

У роботі використовується датасет Adult Dataset з ресурсу UCI Machine Learning Repository.

Робота включає:

* очищення даних (Data Cleaning)
* побудову різних типів графіків
* аналіз залежностей між змінними
* візуалізацію багатовимірних даних

Побудовано наступні графіки:
1. Scatter plot
2. Regression plot
3. Histogram
4. Heatmap
5. Pairplot
6. Countplot
7. Barplot
8. KDE plot

## Вимоги до системи

* Операційна система: Windows / macOS / Linux
* Версія інтерпретатора: Python 3.9+
* Залежності: (`pandas`, `numpy`, `matplotlib` тощо), які вказані у файлі requirements.txt.

## Інструкція із запуску

1. Відкрити термінал у теці lab-3.
2. Створити та активувати віртуальне середовище: `python -m venv venv` `venv\Scripts\activate`
3. Встановити необхідні для виконання роботи залежності командою: `pip install -r requirements.txt`
4. Для виконання роботи потрібно завантажити датасет Adult із сайту https://archive.ics.uci.edu/dataset/2/adult та розташувати у теці lab-3.
5. Запустити Jupyter Notebook в терміналі: `jupyter notebook`
6. Відкрити файл lab3.ipynb та виконайти комірки із кодом.
