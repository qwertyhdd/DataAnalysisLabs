# Лабораторна робота №7: Computer Vision та багатопотоковість у C++

## Опис завдання

Метою лабораторної роботи є розширення попереднього проєкту з OpenCV шляхом інтеграції нейронної мережі для детекції облич та реалізації багатопотокової архітектури для асинхронної обробки відео.

У роботі використовується OpenCV DNN (Deep Neural Networks) та модель ResNet-10 SSD для детекції облич у реальному часі.

Також реалізовано розділення логіки на два потоки:

* основний потік (UI та відображення відео)
* фоновий потік (обробка кадрів нейромережею)

## Реалізований функціонал

### Детекція облич
* Використання нейромережі ResNet-10 SSD
* Завантаження моделі через cv::dnn::readNetFromCaffe
* Обробка кадру через cv::dnn::blobFromImage
* Виявлення облич з confidence > 50%
* Малювання bounding box навколо облич

### Багатопотокова обробка
* Окремий клас FaceDetector
* Використання std::thread для фону обробки
* Захист спільних даних через std::mutex
* Безпечне завершення потоку через std::atomic<bool>

### Архітектура програми
Проект розширює структуру лабораторної №6:
* CameraProvider — отримання відео з камери
* FrameProcessor — обробка кадрів (кольори, фільтри)
* KeyProcessor — обробка клавіш та режимів (включно з FACE mode)
* Display — відображення кадрів
* FaceDetector — нейромережева детекція облич у окремому потоці

### Логіка роботи потоків
#### Main Thread (UI)
* зчитує кадр з камери
* передає кадр у FaceDetector
* отримує останні знайдені координати облич
* малює результати на зображенні
* відображає відео через imshow
#### Worker Thread (FaceDetector)
* очікує новий кадр
* виконує forward pass нейромережі
* обробляє результати детекції
* оновлює спільний буфер координат

### Взаємодія з користувачем

#### Клавіатура
* перемикання режимів обробки (цифри 1-7)
* вихід з програми (клавіша Esc)
* увімкнення детекції обличчя (клавіша F)

#### Миша
* можливість вибору області на зображенні
* відображення прямокутника на кадрі

## Вимоги до системи

* C++ компілятор з підтримкою C++17
* CMake 3.10+
* OpenCV 4.x
