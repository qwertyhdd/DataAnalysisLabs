# Лабораторна робота №4: Візуалізація даних 2

## Опис завдання

Метою лабораторної роботи є створення інтерактивної програми для візуалізації гармонічної функції з накладеним шумом, а також дослідження впливу параметрів сигналу та шуму на результат.

У роботі реалізовано:

* генерацію гармонічного сигналу
* накладання випадкового шуму
* інтерактивне керування параметрами сигналу
* фільтрацію зашумленого сигналу
* порівняння чистого, зашумленого та відфільтрованого сигналів

## Функціональність програми

Програма дозволяє:

* змінювати амплітуду, частоту та фазу гармоніки
* змінювати параметри шуму (середнє значення та дисперсію)
* вмикати/вимикати шум за допомогою чекбоксу
* застосовувати фільтр до сигналу (регулювання через параметр cutoff)
* порівнювати:
    *  чистий сигнал (Clean)
    *  зашумлений сигнал (Noisy)
    *  відфільтрований сигнал (Filtered)
* скидати параметри до початкових за допомогою кнопки Reset

## Вимоги до системи

* Операційна система: Windows / macOS / Linux
* Версія інтерпретатора: Python 3.9+
* Залежності: (`numpy`, `scipy`, `matplotlib`), які вказані у файлі requirements.txt.

## Інструкція із запуску

1. Відкрити термінал у теці lab-4.
2. Створити та активувати віртуальне середовище: `python -m venv venv` `venv\Scripts\activate`
3. Встановити необхідні для виконання роботи залежності командою: `pip install -r requirements.txt`
4. Запустити програму `python lab4.py`.
